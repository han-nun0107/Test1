사용 설명
1. 마우스 휠 or 스페이스 바를 누르시면 옆 페이지로 넘어갈 수 있습니다!

2. 화면 비율은 100%로 맞춰주세요! (ctrl + 마우스 휠)

3. 메인 페이지는 보고 계시는 페이지 포함 총 8개로 준비되어 있습니다.

4. 각 페이지에는 영상 나오는 영어 있는데 클릭 시 영상이 켜집니다.

5. 만약 동영상이 안 나온다면 ctrl + shift + R를 눌러주세요!

6. 영상은 유튜브와 직접 연결한 것으로 저작권 문제는 없습니다. (유튜브 퍼가기 기능 사용)

7. 부족한 실력이지만 잘 봐주셨으면 좋겠습니다! 감사합니다!

8. 마지막으로 🎉생일 축하해🎉 쿼카야!
